How to use Font Awesome Icons?

A few examples:

Font Awesome is a pack with 519 scalable vector icons. Check out Font Awesome Website for more information.

Here is an example of the code to display an icon: 


<i class="fa fa-bar-chart"></i>

How to use Glyphicons?

A few examples:

Contains a total of 260 icons. Check out Bootstrap Website for more information.

Here is an example of the code to display an icon: 


<span class="glyphicon glyphicon-envelope" aria-hidden="true"></span>

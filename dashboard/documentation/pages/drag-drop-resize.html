Drag, Drop & Resize

Quarca uses jQuery UI and an amazing plugin called GRIDSTACK.JS for draggable, droppable and resizable widgets.


How to use it?

Step 1: Include the plugin's JS and jQuery UI after the wrapper div/before the body ending tag.


<script src="vendor/plugins/others/underscore/underscore-min.js"></script>

<script src="vendor/jqueryui/jquery-ui.min.js"></script>

<script src="vendor/plugins/ui/gridstack/gridstack.min.js"></script>

Step 2: Add the HTML and initialize the plugin.

Columns

There are 5 types of columns available. The plugin/widgets requires 2 important things to work;
1. Specify the column type in the second div's class after grid-stack. eg. cols-2, cols-3...
2. Specify the width when initializing the plugin.. eg. in the js width: 2 for 2 columns.

Widget Sizes

1. Width

The widget's width is set by the number of columns that you want it to fill.
For example if you are using the 4 columns grid and want the widget to be 25%(1/4) of it.
Set the data-gs-width to data-gs-width="1".

2 columns; the data-gs-width will be from 1 to 2.
3 columns; the data-gs-width will be from 1 to 3.
4 columns; the data-gs-width will be from 1 to 4.
5 columns; the data-gs-width will be from 1 to 5.
6 columns; the data-gs-width will be from 1 to 6.


2. Height

The widget's height is specified in the initialization of the plugin as cell_height: 110 or any number.

This means that if you set the data-gs-height to 1 it will be equal to 110px.
If set to 2, it will be equal to 220px.

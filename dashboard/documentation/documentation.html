Introduction

Quarca is a powerful admin dashboard template based on the latest Twitter Bootstrap framework. Built with HTML5, CSS, jQuery and LESS, this template can be used for whatever purpose you want; be it an amazing admin panel to control your website/app or a high-end web application. Each pixel has been carefully crafted to look fantastic and has been optimized to allow unlimited customization.

Getting started

Quarca was built on Twitter Bootstrap 3.3.4 and every component can still be used even if they don't all appear in the template. The core files have not been modified, this means that we can easily update to newer Bootstrap versions in the future.



File Structure

See the next topic for a tree view of the file structure. It is basically divided into two parts, ASSETS and VENDOR.

ASSETS:
Assets contain files that have been created from scratch, specifically for the template(CSS, JS, Images...). Everything that you create or do by yourself should go into assets/.

VENDOR:
Vendor contains third party fonts, javascript libraries, plugins and the Bootstrap framework. If you are adding a new javascript library or plugin it should go straight into vendor/.

ROOT FOLDER:
Contains all HTML or PHP files.



Required Files

There is a bunch of files that are extremely important for Quarca to function properly.


CSS(Goes inside of the head tag):

  1. vendor/bootstrap/bootstrap.min.css - Bootstrap.

  2. assets/css/style.css - Quarca/structure.

  3. assets/css/ui.css - UI elements.

  4. assets/css/themes/theme-default.css - Theme styling compiled from LESS.


Fonts(Goes inside of the head tag):

  1. <link href='http://fonts.googleapis.com/css?family=Open+Sans:300,400,600' rel='stylesheet' type='text/css'>
    Open Sans, Main font for headings and content. *Can be changed to any font.

  2. vendor/fonts/font-awesome.min.css - Font Awesome, icon webfont.


JS(Goes after the wrapper div/before the body ending tag):

  1. vendor/js/required.min.js - Important JS files like;
    jQuery Library, Bootstrap, jQuery Mousewheel, jScrollPane, metisMenu, Moment.js, jquery.matchHeight.js and jquery Cookies

  2. assets/js/quarca.js - Important JS for Quarca to work.



Quarca's Structure

Left and Right sidebar are available. Left sidebar is set by default.
To use the right sidebar add right-sidebar to the body tag class.



Theming

Quarca uses LESS for coloring purposes only. LESS extends CSS with dynamic behaviors such as variables, check the website for more information.
9 themes are available. But it is possible to create unlimited themes by simply changing variables' colors when editing the LESS file. Change the color on a line or two and you have a new theme. Variables are commented on the LESS files. Theme files are located here: assets/less/. Use editors like Crunch to edit and compile LESS to CSS.

How to use buttons?

Buttons can easily be inserted anywhere on the page by inserting the codes provided below.
Check out the "HTML" tab to get the code of any button.


How to use different sizes?

By simply adding btn-lg, btn-sm or btn-xs in the button class.
To keep the default button size(as in the demo below), do not add the size classes.

To use different colors, change some class names. Button dropdown is available in all the colors shown above.
Here is how to do it:
On line 2 & 3, replace btn-default by btn-primary or by any other color.
On line 7, replace default by primary or by any other color. Do not include btn at the beginning of the class name. 


<div class="btn-group">

    <button type="button" class="btn btn-default">Default</button>

    <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown" aria-expanded="false">

        <i class="fa fa-arrow"></i>

        <span class="sr-only">Toggle Dropdown</span>

    </button>

    <ul class="dropdown-menu default" role="menu">

        <li><a href="#">Link 1</a></li>

        <li><a href="#">Link 2</a></li>

        <li><a href="#">Link 3</a></li>

        <li class="divider"></li>

        <li><a href="#">Separated Link</a></li>

    </ul>

</div>

To use different colors, change some class names. Button group is available in all the colors shown above.
Here is how to do it:
On line 3, 11 & 12, replace btn-default by btn-primary or by any other color.
On line 7, replace default by primary or by any other color. Do not include btn at the beginning of the class name.

If no dropdown/up is required, remove the btn-group div (from line 2 to 10). 


<div class="btn-group quarca" role="group" aria-label="...">

    <div class="btn-group" role="group">

        <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown" aria-expanded="false">

            Drop

            <span class="caret"></span>

        </button>

        <ul class="dropdown-menu default" role="menu">

            <li><a href="#">Link</a></li>

        </ul>

    </div>

    <button type="button" class="btn btn-default">Middle</button>

    <button type="button" class="btn btn-default">Right</button>

</div>

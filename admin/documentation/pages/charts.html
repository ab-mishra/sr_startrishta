Flot Charts

More information: Flot Charts

Step 1: Add the JS files after the wrapper div. 


<!--[if lte IE 8]><script language="javascript" type="text/javascript" src="vendor/plugins/charts/excanvas.min.js"></script><![endif]-->

<!-- Flot -->

<script src="vendor/plugins/charts/flot/jquery.flot.min.js"></script>

<!-- Pie -->

<script src="vendor/plugins/charts/flot/jquery.flot.pie.min.js"></script>

<!-- Responsive -->

<script src="vendor/plugins/charts/flot/jquery.flot.resize.min.js"></script>

<!-- Tooltip -->

<script src="vendor/plugins/charts/flot/jquery.flot.tooltip.min.js"></script>

<!-- Real Time -->

<script src="vendor/plugins/charts/flot/jquery.flot.time.min.js"></script>

<!-- Init -->

<script src="assets/js/init/init.flot.js"></script>

Step 2: Add the HTML & edit init.flot.js to initialize charts.

Flot HTML example: 


<div class="flot-chart">

    <div id="line-chart" class="flot line"></div>

</div>

Important
1. Make sure to wrap the chart inside of a div class named flot-chart.
2. Make sure that the chart div has the class flot + line, bar or pie.


Chart.js

More information: Chart.js

Step 1: Add the JS files after the wrapper div. 


<script src="vendor/plugins/charts/chartjs/Chart.min.js"></script>

<script src="assets/js/init/init.chartjs.js"></script>

Step 2: Add the HTML & edit init.chartjs.js to initialize charts.

Chart.js HTML example: 


<div class="chartjs">

    <div class="chartjs-container">

        <canvas id="line-chart" width="1000" height="400"></canvas>

    </div>

</div>

Important
1. Make sure to wrap the chart inside of a first div class named chartjs and a second one called chartjs-container
2. Make sure that the chart has a size. Height is important.


Easy Pie Chart

More information: EASY PIE CHART

Step 1: Add the JS files after the wrapper div. 


<script src="vendor/plugins/charts/easypie/jquery.easypiechart.min.js"></script>

<script src="assets/js/init/init.easypie.js"></script>

Step 2: Add the HTML & edit init.easypie.js to initialize charts.

Easy Pie Chart HTML example: 


<div id="demo1" class="easy-chart chart-xs" data-percent="18">18%</div>

How to use different sizes?

By adding chart-xs, chart-sm, chart-md or chart-lg to the div's class.
